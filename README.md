# Лабораторные работы по курсу "Информационный Поиск"

| Студент                | Группа      | Оценка |
| ---------------------- | ----------- | ------ |
| Кочев Дмитрий Олегович | М8О-406Б-22 |        |

Проект реализует полнотекстовую поисковую систему для корпуса статей о Формуле 1 с сайтов:
- **f1news.ru**
- **f1report.ru**

Система включает полный цикл обработки: от сбора данных и индексации до ранжирования результатов и анализа распределения слов.

## Структура проекта

    ├── lab1/      # Сбор корпуса документов
    ├── lab2/      # Поисковый робот
    ├── lab3-5/    # Токенизация текста, анализ закона Ципфа, стемминг
    ├── lab6/      # Построение булева индекса
    ├── lab7/      # Булев поиск
    └── report/    # Отчет в LaTeX
